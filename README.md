# Behavior-Based Ransomware Detection

This project implements a behavior-based ransomware detection system using deep learning (BiLSTM + Attention) on behavioral features.

## Features
- Sequence modeling using BiLSTM
- Attention mechanism for interpretability
- Behavioral feature analysis
- Binary classification (Benign vs Ransomware)

## Tech Stack
- Python
- PyTorch
- Scikit-learn
- NumPy, Pandas

## Dataset
RanSMAP 2024 behavioral dataset (Kaggle)
